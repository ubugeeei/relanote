# Relanote Examples

## Tutorials

Step-by-step tutorials to learn Relanote syntax and features.

| File | Topic | Description |
|------|-------|-------------|
| [01_hello.rela](./tutorials/01_hello.rela) | Hello World | Basic scale definition and melody |
| [02_relative_rhythm.rela](./tutorials/02_relative_rhythm.rela) | Relative Rhythm | Notes divided equally within blocks |
| [03_block_concatenation.rela](./tutorials/03_block_concatenation.rela) | Block Concatenation | Combining blocks with `++` operator |
| [04_articulations.rela](./tutorials/04_articulations.rela) | Articulations | Staccato (`*`), accent (`^`), portamento (`~`) |
| [05_transformations.rela](./tutorials/05_transformations.rela) | Transformations | Transpose, reverse, repeat with `\|>` |
| [06_chords.rela](./tutorials/06_chords.rela) | Chords | Chord syntax `[]` and progressions |
| [07_tuplets.rela](./tutorials/07_tuplets.rela) | Tuplets | Tuplet syntax `{notes}:n` for triplets etc. |
| [08_microtones.rela](./tutorials/08_microtones.rela) | Microtones | Semitone modifiers `+` and `-` |
| [09_synth_basics.rela](./tutorials/09_synth_basics.rela) | Synth Basics | Preset synths and parameter control |
| [10_chiptune.rela](./tutorials/10_chiptune.rela) | Chiptune | 8-bit presets (NES, GameBoy, etc.) |
| [11_synth_advanced.rela](./tutorials/11_synth_advanced.rela) | Advanced Synth | Custom synth definitions |
